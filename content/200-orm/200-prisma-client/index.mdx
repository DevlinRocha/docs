<BoxTitle>Prisma Client</BoxTitle>

Prisma Client is a powerful, auto-generated query builder that makes working with your database a breeze. It provides:

- **Type safety**: Leverage TypeScript's static typing for database queries
- **Auto-completion**: Enjoy IDE support for an improved developer experience
- **Intuitive API**: Write database queries using a clear and concise API
- **Relation handling**: Easily work with related data across tables

</BorderBox>
